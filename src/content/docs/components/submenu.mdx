---
title: Submenu
description: Accordion-style submenu component for navigation menus with mobile-friendly toggle functionality.
---
import { Tabs, TabItem } from '@astrojs/starlight/components';
import SubMenu from '@components/submenu/Submenu.astro';

The Submenu component provides accordion-style navigation for nested menu items, ideal for mobile menus or compact navigation structures.

## Props
| Prop Name    | Type            | Default       | Description                                         |
|--------------|-----------------|---------------|-----------------------------------------------------|
| items        | Array           | []            | Array of menu items with optional nested submenus.    |
| className    | String          | ""            | Additional CSS classes for custom styling.              |
